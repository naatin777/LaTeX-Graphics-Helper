# Safe Mode仕様

## 目的

変換結果の出力先に既存ファイルがある場合の動作を明示し、意図しない上書きを防ぐ。

## 状態

- 初期値はONとする。
- status barでON/OFFを確認し、切り替えられる。

## Safe Mode ON

1回の変換で競合する出力が1件以上ある場合、出力反映前に確認を1回だけ表示する。

選択肢は「両方残す」「上書きしない」「上書きする」とする。独立した「キャンセル」は表示しない。ダイアログを閉じた場合は「上書きしない」と同じく全体停止する。

選択は、その変換に含まれるすべての競合出力へ適用する。

### 両方残す

既存ファイルを変更せず、変換結果に拡張子直前の連番suffixを付ける。存在しない最小の正整数を使用する。

### 上書きしない

指定出力先へ何も反映せず、変換全体を停止する。

### 上書きする

既存ファイルを保護したうえで変換結果を反映する。反映途中で失敗した場合は、既存ファイルを復元する。

## Safe Mode OFF

確認を表示せず上書きを適用する。失敗時の復元とUndoのため、既存ファイルは保護する。

## Undo

新規作成・両方残す・上書きのいずれも、成功後の直前変換として[Undo仕様](undo-last-conversion.md)に従う。
