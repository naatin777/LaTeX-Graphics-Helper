## Format-specific boundaries

- `native Draw.io document`: `.drawio`または`.dio`のauthoring document。pageごとのPDFまたは1つのmulti-page PDFへ外部toolでexportする。
- `editable Draw.io image`: `.drawio.png`、`.dio.png`、`.drawio.svg`、`.dio.svg`。通常のformat conversionからDraw.io adapterへrouteする。
- `page PDFs`: native Draw.ioの各pageを個別PDFにした結果。`outputPath.convertDrawioToPdf`と`${page}`を使う既存surfaceを維持する。
- `single PDF`: native Draw.ioの全pageを1つにした結果。`outputPath.convertDrawioToPdfDirectly`を使う既存surfaceを維持する。`directly`は新しい内部名に使わない。
